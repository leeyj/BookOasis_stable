# 📱 OPDS 모바일 뷰어 연동 가이드 (OPDS Guide)

이 문서는 모바일 및 태블릿 환경의 외부 리더 애플리케이션(앱)에서 **BookOasis OPDS 피드**를 통해 도서 보관함에 무선 접속하고, 책을 스트리밍하거나 다운로드하는 방법을 설명합니다.

---

## 1. OPDS란 무엇인가요?
**OPDS (Open Publication Distribution System)**는 전자책 및 만화책 카탈로그를 배포하기 위한 Atom XML 기반의 업계 표준 프로토콜입니다. 
BookOasis는 OPDS 서버 기능을 기본 내장하고 있어, 전용 웹 뷰어 외에도 스마트폰이나 태블릿의 **다양한 외부 독서 앱에서 보관함의 시리즈 및 책 목록을 탐색하고 즉시 다운로드**할 수 있도록 연동을 지원합니다.

---

## 2. BookOasis OPDS 엔드포인트 URL

외부 앱에 등록해야 하는 BookOasis OPDS 카탈로그 주소는 다음과 같습니다. 
접속을 시도할 때 **BookOasis 웹 서재에 로그인할 때 사용하는 아이디와 비밀번호**를 통한 Basic Authentication(기본 인증)이 필수적으로 요구됩니다.

### 🌐 일반 도서 OPDS 카탈로그 주소
> `http://<서버_IP_또는_도메인>:5930/opds`
* **접속 권한**: BookOasis에 등록된 모든 일반 유저 계정으로 접근 가능.
* **제공 카테고리**:
  * 라이브러리 목록별 시리즈 탐색
  * 신규 추가 도서 (Recently Added)
  * 최근 읽은 도서 (Recently Read)
  * 통합 검색 기능 (제목, 시리즈명, 저자 검색 지원)

### 🔞 성인 도서 OPDS 카탈로그 주소
> `http://<서버_IP_또는_도메인>:5930/opds-adult`
* **접속 권한**: **관리자(Admin) 권한**을 가진 계정으로만 접근 및 인증 가능.
* **제공 카테고리**: 성인 도서 보관함의 시리즈, 도서 카탈로그 및 통합 검색 기능 제공.

### 📖 타치요미 / 미혼 (Tachiyomi / Mihon) 전용 주소
> 일반: `http://<서버_IP_또는_도메인>:5930/app-opds`
> 성인: `http://<서버_IP_또는_도메인>:5930/app-opds-adult`
* 타치요미나 미혼 등 특수 앱에서 연동할 때 사용하는 전용 엔드포인트입니다. 표준 OPDS 형식에 맞게 앱에서 원활하게 파싱할 수 있도록 최적화되어 있으며 통합 검색 기능도 제공합니다.

### 🔍 OPDS 검색 엔드포인트 (OpenSearch 지원)
BookOasis OPDS 피드는 OpenSearch 규격을 지원하므로 지원하는 외부 독서 앱에서 키워드를 통한 도서 검색이 가능합니다. (제목, 시리즈명, 저자명 매칭)
* **일반 검색 주소**: `http://<서버_IP_또는_도메인>:5930/opds/search?q={searchTerms}`
* **성인 검색 주소**: `http://<서버_IP_또는_도메인>:5930/opds-adult/search?q={searchTerms}`
* **타치요미 일반 검색 주소**: `http://<서버_IP_또는_도메인>:5930/app-opds/search?q={searchTerms}`
* **타치요미 성인 검색 주소**: `http://<서버_IP_또는_도메인>:5930/app-opds-adult/search?q={searchTerms}`
> ※ 대부분의 모던 OPDS 앱(문리더, KyBook 등)은 상위 카탈로그 주소만 등록하면 자동으로 검색 API 주소를 감지해 연동합니다.

## 3. 대표적인 모바일 리더 앱 연동 방법

OPDS 카탈로그 기능을 지원하는 대표적인 외부 뷰어 앱의 설정 방법입니다.

### 🍎 iOS (아이폰 / 아이패드)
* **KyBook 3 / Yomu / PocketBook Reader** 등
  1. 앱을 실행하고 **"카탈로그(Catalog)"** 또는 **"OPDS 책장 추가"** 메뉴로 진입합니다.
  2. 새로운 카탈로그 추가(+) 버튼을 누릅니다.
  3. **카탈로그 주소**에 `http://<서버_IP>:5930/opds`를 입력합니다.
  4. 인증 정보 입력 창이 활성화되면, BookOasis 로그인용 **아이디**와 **비밀번호**를 입력합니다.
  5. 추가가 완료되면 BookOasis의 보관함들이 목록으로 노출되며, 책을 터치하여 바로 다운로드 및 독서를 시작할 수 있습니다.

### 🤖 Android (안드로이드 폰 / 태블릿)
* **Moon+ Reader (문+ 리더) / FBReader / Aldiko** 등
  1. 앱의 좌측 사이드바 메뉴에서 **"네트워크 라이브러리(Net Library)"**를 선택합니다.
  2. **"새 책장 추가(Add new catalog)"**를 클릭합니다.
  3. 책장 이름(예: `BookOasis`)을 입력하고, 주소란에 `http://<서버_IP>:5930/opds`를 기입합니다.
  4. 웹 로그인 계정정보(사용자명, 비밀번호)를 입력한 뒤 저장합니다.
  5. 등록된 책장에 들어가 도서 목록을 자유롭게 다운로드하여 즐길 수 있습니다.

* **타치요미 (Tachiyomi) / 미혼 (Mihon)**
  1. **확장 기능(Extensions)** 탭에서 OPDS용 확장 프로그램(예: `Kavita`나 일반 `OPDS` 확장 기능)을 설치합니다.
  2. **소스(Sources)** 탭에서 추가된 OPDS 확장의 설정을 엽니다.
  3. 서버 주소란에 반드시 전용 엔드포인트인 `http://<서버_IP>:5930/app-opds` (성인용은 `/app-opds-adult`)를 기입합니다.
  4. 사용자 이름 및 비밀번호를 설정한 뒤 저장하고 탐색(Browse)을 시작합니다.

## 4. 지원 포맷 및 주요 안내 사항

* **지원 확장자**: `EPUB`, `ZIP`, `CBZ`, `PDF`, `TXT` 등 보관함에 스캔된 모든 도서 파일 포맷을 전송합니다.
* **외부망 연동 (외부에서 접속 시)**:
  * 공유기 환경의 홈 서버인 경우, 공유기 관리 페이지에서 **5930 포트**를 포트포워딩 설정하시거나, 리버스 프록시(Nginx 등) 및 DDNS 설정을 해 두셔야 집 밖에서도 LTE/5R 모바일 데이터망을 통해 접속하실 수 있습니다.
* **비밀번호 보안**:
  * 외부 OPDS 연결 통신은 Basic Auth를 사용하므로, 보안 강화를 위해 Nginx 리버스 프록시에 **SSL 인증서(HTTPS)**를 설정하여 도메인 접속 주소를 통한 통신을 암호화하여 연동하시는 것을 적극 권장합니다.
