라이브러리 자동 스캔 & 스케줄 설정

각 카테고리의 동기화 스캔 스케줄(Cron) 주기를 일괄 관리하고 즉시 실행할 수 있습니다.

카테고리명 서버 물리 경로 Rclone RC 주소 스케줄 주기 (Cron식) 최근 스캔 시각 상태 작업

기본 시스템 설정

북오아시스 미디어 서버의 전반적인 환경설정을 직접 관리할 수 있습니다.

보관함의 도서 커버 크기를 120px ~ 240px 범위 내에서 본인 모니터 해상도에 최적화하여 슬라이더로 크기를 조절할 수 있습니다.

무한 스크롤 및 보관함 로딩 시 한 화면에 렌더링할 도서 카드의 단위 개수입니다.

동시 접속량이 증가할 때 DB 연결 자원을 재사용하는 크기입니다. SQLite 권장 설정은 5~10개이며, 최대 20개까지 제한됩니다.

스캐너 백그라운드 구동 시 생성되는 로그의 상세 출력 방식을 정의합니다. 비활성화 시 로그 생성 부하와 파일 크기가 극적으로 줄어듭니다.

구글 드라이브 마운트 캐시 갱신을 위해 스캐너가 접근할 rclone RC 서버 주소입니다. 쉼표(,)로 구분하여 복수의 주소를 등록할 수 있습니다. (예: http://localhost:5572, http://localhost:5573)

이미지 로드 시작 후 로딩 오버레이가 나타날 때까지 대기할 지연 시간입니다. 외부 접속(클라우드플레어 등) 시 500ms 이상으로 지연을 설정하는 것을 권장합니다. (브라우저 로컬 저장)

표지가 유실되었거나 원격지 PDF 등 스캔 시 제외된 도서들의 표지를 1권씩 3초 간격으로 추출하는 독립 스캐너의 스케줄 주기입니다. (기본값: 매일 새벽 3시 `0 3 * * *` 권장)

사이드바 '최근 읽은 도서' 메뉴에서 보여줄 최대 도서 개수입니다. (기본 30개)

스캐너 동작 중 시스템 전체 가용 메모리가 이 값(MB) 미만으로 떨어지면 스캔 작업을 일시 중단하여 시스템 뻗음을 방지합니다. (기본 1536MB)

스캐너 자체 프로세스가 소모하는 메모리가 이 값(MB)을 초과할 경우 안전하게 스캔을 종료 및 재시작하여 메모리 누수를 방지합니다. (기본 2048MB)

[경고] 리버스 프록시(Authentik, Authelia 등)가 앞단에 구축된 홈랩 환경에서만 활성화하세요. 이 옵션을 켜면 `Remote-User` 등 HTTP 헤더의 유저명을 신뢰하여 비밀번호 없이 세션을 강제 생성합니다. 외부망에 직접 노출된 상태로 켤 경우 심각한 보안 취약점이 됩니다!
