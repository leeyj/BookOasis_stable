최고의 독서 환경을 위한 기능들

과도한 프레임워크나 빌드 도구를 걷어내고, 오직 브라우저와 홈 서버 본연의 속도에 초점을 맞추었습니다.

초고속 로컬 스캐너

백그라운드에서 만화책 압축파일(CBZ, ZIP, PDF)의 볼륨 크기와 오프셋을 밀리초 단위로 스캔하여 최적의 스트리밍 상태를 준비합니다.

장르 & 태그 클라이언트 필터링

서버 요청을 최소화하고, 클라이언트에 로드된 모든 시리즈 데이터를 기반으로 태그와 장르를 딜레이 없이 즉각 필터링합니다.

안전한 OPDS 서버 제공

Chunky Reader, Panels 등 외부 모바일 앱 런처와 로그인 세션 기반의 안전한 OPDS 연동을 지원하여 어디서든 서재를 연동할 수 있습니다.

Pure Vanilla JS & CSS

리액트, 뷰와 같은 무거운 라이브러리 빌드 단계 없이 순수한 HTML, CSS, JavaScript로 브라우저 자원을 100% 효율적으로 사용합니다.
